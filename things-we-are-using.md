FRONTEND
└── React (dashboard UI)
└── Three.js (animations/styling)
└── Axios (talks to backend)

BACKEND
└── Node.js + Express (the manager)
└── JWT + Bcrypt (security)
└── Nodemailer (sends emails)
└── Handlebars (email templates)
└── BullMQ (queue manager)

STORAGE
└── MongoDB (all your data)
└── Redis (BullMQ's notepad)

EXTERNAL
└── AWS SES (email delivery pipe)
